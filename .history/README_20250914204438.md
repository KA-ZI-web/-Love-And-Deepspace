## 项目简介
### 一个基于 React 的游戏角色（沈星回）交互式展示页，包含首页，角色介绍，特色玩法等模块。
----------------------------------------------
## 技术栈
### 核心前端框架：React+Ts
### 路由管理：ReactRouter
### UI组件：Swiper+ReactTooltip
### 构建与工程化工具：CRA生态+pnpm+Webpack
--------------------------------------
## 快速开始
### 安装依赖
### pnpm install
### 本地启动
### pnpm start
-------------------------
